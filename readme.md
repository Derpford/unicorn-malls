# Unicorn Malls

A game about the eldritch horror of nostalgiabaiting.

## How To Use These Rules
- main.md holds the rules you need to get started as a player, as well as descriptions of various Items and Psyches you might use and a brief overview of what the Malls even are.
- bestiary.md holds a list of Shades you might encounter in the Malls. This is spoilers, so only read it if you're going to GM!
- spoilers.md holds setting details that players aren't supposed to know right away, such as the true nature of the Malls.
